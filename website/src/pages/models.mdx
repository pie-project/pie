# Models

What you can run on Pie depends on three choices: the model architecture, the
driver Pie routes inference through, and the precision and parallelism that
driver supports for that architecture. This page is the at-a-glance matrix.
For configuration and supported architectures of each driver, see
[CUDA](/docs/reference/drivers/cuda) and [Portable](/docs/reference/drivers/portable).
