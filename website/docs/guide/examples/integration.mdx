# I/O and integration

Inferlets that talk to the outside world: HTTP servers, HTTP clients, MCP servers, the OpenAI Responses protocol, and SDK concurrency tests.

## HTTP and embedded servers

| Inferlet | What it shows |
|---|---|
| [`http-server`](https://github.com/pie-project/pie/tree/main/inferlets/http-server) | `wasi:http/incoming-handler` server with SSE streaming. |
| [`image-fetch`](https://github.com/pie-project/pie/tree/main/inferlets/image-fetch) | `wstd::http::Client` to fetch and decode images. |
| [`openresponses`](https://github.com/pie-project/pie/tree/main/inferlets/openresponses) | OpenResponses-compliant HTTP API. |
