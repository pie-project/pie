# Structured output

Inferlets that constrain the model's output to a grammar, a JSON Schema, or a templated form. The same idea in three languages.

| Inferlet | What it shows |
|---|---|
| [`constrained-decoding`](https://github.com/pie-project/pie/tree/main/inferlets/constrained-decoding) | Grammar-constrained decoding with a Lark grammar. |
| [`json-schema-validation`](https://github.com/pie-project/pie/tree/main/inferlets/json-schema-validation) | JSON generation constrained against a runtime-string JSON schema. |
| [`template-generation`](https://github.com/pie-project/pie/tree/main/inferlets/template-generation) | Grammar-constrained JSON merged with a Jinja2 template. |
| [`python-constrained-decoding`](https://github.com/pie-project/pie/tree/main/inferlets/python-constrained-decoding) | Same idea in Python. |
| [`js-constrained-decoding`](https://github.com/pie-project/pie/tree/main/inferlets/js-constrained-decoding) | Same idea in JavaScript. |
