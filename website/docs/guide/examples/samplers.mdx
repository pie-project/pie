# Sampling and search

Inferlets that transform or search model distributions during decoding.

| Inferlet | What it shows |
|---|---|
| [`sampling-primitives`](https://github.com/pie-project/pie/tree/main/inferlets/sampling-primitives) | Token, logits, entropy, probabilities, and log-probabilities from one forward pass. |
| [`consensus-decoding`](https://github.com/pie-project/pie/tree/main/inferlets/consensus-decoding) | Parallel candidate generation with string-similarity consensus ranking. |
| [`greenlist-watermarking`](https://github.com/pie-project/pie/tree/main/inferlets/greenlist-watermarking) | Greenlist token watermarking during generation. |
| [`mirostat-v2-sampling`](https://github.com/pie-project/pie/tree/main/inferlets/mirostat-v2-sampling) | Adaptive-surprise sampling with Mirostat v2. |
| [`contrastive-decoding`](https://github.com/pie-project/pie/tree/main/inferlets/contrastive-decoding) | Same-model contrastive decoding with full-context expert and bounded-context amateur passes. |
| [`beam-search`](https://github.com/pie-project/pie/tree/main/inferlets/beam-search) | On-device beam expansion over a shared prefix-tree KV cache. |
